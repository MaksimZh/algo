Оценка сложности алгоритма.

Расчёт зависимости энергии электрона от импульса
в периодической полупроводниковой структуре методом Фурье.

Решается система линейных дифференциальных уравнений вида
d/dz [A(p, z) df/dz] + B(p, z) df/dz + C(p, z) f = E f;
где
f - вектор неизвестных функций размерности D;
E - неизвестная энергия;
p - вектор импульса, который задаётся как параметр;
A, B, и C - матрицы размерности DxD, которые при каждом значении p
являются периодическими, кусочно-постоянными функциями z.

Параметры входных данных:
D - размерность системы уравнений;
N - количество слоёв в одном периоде (A, B, и C постоянны внутри каждого слоя);
K - размерность базиса (количество членов в ряде Фурье, которые мы учитываем);
M - количество разных значений импульса, для которых вычисляется энергия.

Расчёт происходит в 4 этапа:

1. Вычисление коэффициентов в уравнении.
Все коэффициенты вычисляются отдельно для каждого слоя.
Сложность пропорциональна количеству слоёв внутри одного периода (N)
и количеству коэффициентов в уравнении (3 * D * D).
Следовательно, сложность данного этапа: O(D^2 * N).

2. Применение преобразования Фурье.
От всех коэффициентов нужно взять интеграл с каждой парой элементов базиса
в каждом слое внутри периода.
Количество коэффициентов равно 3 * D * D, число слоёв равно N,
а количество пар элементов базиса равно K * K.
Следовательно, сложность данного этапа: O(D^2 * K^2 * N).

3. Подстановка в уравнения конкретных значений импульса.
После применения преобразования Фурье количество слоёв уже не важно.
Вместо дифференциального уравнения теперь мы имеем уравнение
на собственные значения матрицы, размерности (D * K) x (D * K),
каждый элемент которой зависит от импульса.
Количество коэффициентов равно (D * K)^2, а число разных значений импульса равно M.
Следовательно, сложность данного этапа: O(D^2 * K^2 * M).

4. Вычисление собственных значений матриц.
Сложность вычисления всех собственных значений квадратной матрицы
в общем случае пропорцинальна кубу размерности этой матрицы, то есть (D * K)^3.
Такую операцию нужно проводить для каждого значения импульса, то есть M раз.
Следовательно, сложность данного этапа: O(D^3 * K^3 * M).

Сложность всех этапов вместе: O(D^2 * N) + O(D^2 * K^2 * N) + O(D^2 * K^2 * M) + O(D^3 * K^3 * M).
При асимптотической оценке остаются наиболее быстро растущие слагаемые: O(D^2 * K^2 * N) + O(D^3 * K^3 * M).
Как правило, в таких задачах N << M, поэтому, окончательный ответ: O(D^3 * K^3 * M).
