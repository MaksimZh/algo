Оценка сложности алгоритма
Расчёт зависимости энергии электрона от импульса
в периодической полупроводниковой структуре
методом Фурье

Параметры входных данных:
N - количество слоёв в одном периоде
B - размерность базиса (количество членов в ряде Фурье)
M - количество элементов сетки в пространстве импульсов

Расчёт происходит в 4 этапа:

1. Вычисление коэффициентов в уравнении в виде кусочно-постоянных функций.
Сложность пропорциональна количеству отрезков внутри одного периода (N).
Следовательно, сложность данного этапа: O(N).

2. Применение преобразования Фурье к искомой функции и уравнениям.
От всех коэффициентов в уравнении на каждом отрезке внутри периода
нужно взять интеграл с каждой парой элементов базиса.
Количество отрезков равно N, а количество пар элементов базиса равно B*B.
Следовательно, сложность данного этапа: O(B^2 * N).

3. Подстановка в уравнения конкретных значений импульса.
После применения преобразования Фурье количество отрезков уже не важно.
Вместо дифференциального уравнения теперь мы имеем уравнение на собственные 
значения матрицы, размерности BxB, каждый элемент которой зависит от импульса.
Количество коэффициентов равно B*B, а число разных значений импульса равно M.
Следовательно, сложность данного этапа: O(B^2 * M).

4. Вычисление собственных значений матриц.
Сложность вычисления всех собственных значений квадратной матрицы
в общем случае пропорцинальна кубу размерности этой матрицы, то есть B^3.
Такую операцию нужно проводить для каждого значения импульса, то есть M раз.
Следовательно, сложность данного этапа: O(B^3 * M).

Сложность всех этапов вместе: O(N) + O(B^2 * N) + O(B^2 * M) + O(B^3 * M).
Асимптотическая оценка: O(B^2 * N) + O(B^3 * M).
Как правило, в таких задачах N << M, поэтому, окончательный ответ:
O(B^3 * M)
